# Electon webpack typescript

## Versions

- node `12.16.1`
- realm `3.7.0-alpha.2`
- electron `7.1.14`

## Development

```bash
# load correct node version
nvm use

# install dependencies
yarn

# local development
yarn dev
```

## Distribute

```bash
yarn package
```
